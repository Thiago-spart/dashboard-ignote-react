## Requirements

Before starting üèÅ, you need to have [Git](https://git-scm.com), [Node](https://nodejs.org/en/) and [Yarn](https://yarnpkg.com) installed, and an account on [google api](https://cloud.google.com/google/api) to use google as the auth provider, if not you can choose any provider you want.

also you have to fill the .env file.

### Starting

```bash
# Clone this project
$ git clone git@github.com:Thiago-spart/dashboard-ignote-react.git

# Access
$ cd dashboard-ignote-react

# Install dependencies
$ yarn

# Run the project in developer mode
$ yarn dev

# The server will initialize in the <http://localhost:3000>
```
